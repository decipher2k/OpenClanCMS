*The ClanSphere Readme*<br>
<br>
-------------------------------------------------------------------------------<br>
<br>
ClanSphere is a web portal software for clans, guilds, but also other groups.<br>
<br>
It is developed in PHP and runs on Apache, Lighttpd, Microsoft IIS and Nginx.<br>
<br>
Data storage requires one of MySQL, PostgreSQL, SQLite or Microsoft SQLSRV.<br>
<br>
The current version is available under the terms of the 'New BSD License'.<br>
<br>
Our project website is located at http://www.clansphere.net<br>
<br>
Look into '/docs' and '/webserver' for further documentation.
